# Sales-Data-Cleaning-Analysis-Project-Excel-SQL-Pandas-Power-BI-
This project focuses on real-world data cleaning and analysis using multiple data analytics tools. The main objective of this project was to transform raw and messy data into a clean, structured dataset and then perform analysis to generate business insights.
🛠 Tools & Technologies Used

Microsoft Excel – Data cleaning and preprocessing

MySQL – Data querying and analysis

Python (Pandas) – Data manipulation and exploratory analysis

Power BI – Dashboard creation and business storytelling

🧹 Data Cleaning Process (Excel)

The dataset contained several issues that were cleaned and standardized using Excel:

Standardized inconsistent city and category names

Converted date formats into proper date type

Cleaned discount column and converted percentage values into decimal format

Handled missing values

Removed duplicate records

Standardized column naming conventions

Created a Revenue column using the formula:

Revenue = Quantity × Price × (1 − Discount)

These steps ensured the dataset became analysis-ready and consistent across tools.

📊 Data Analysis (SQL & Pandas)

After cleaning the dataset, analysis was performed using SQL and Python.

Key Analysis Tasks

Total revenue calculation

Revenue by city

Revenue by product category

Customer purchase patterns

Payment method distribution

Identification of top performing customers

📈 Power BI Dashboard

An interactive dashboard was built to visualize the insights.

Dashboard KPIs

Total Revenue

Total Orders

Average Order Value

Visualizations

Revenue by City

Revenue by Product Category

Payment Method Distribution

Monthly Sales Trends

Top Customers Table

The dashboard helps stakeholders quickly understand sales performance and customer behavior.

📖 Key Insights

Some important insights generated from the analysis include:

Identification of the top revenue-generating city

Most profitable product categories

Most commonly used payment methods

Top customers contributing significantly to total revenue

These insights demonstrate how cleaned data can be transformed into actionable business intelligence.

🎯 Project Goal

The goal of this project was to simulate the real workflow of a data analyst, including:

Raw Data → Data Cleaning → Data Analysis → Dashboard Storytelling

This project helped strengthen skills in data preprocessing, analytical thinking, and business communication.
